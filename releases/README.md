# Release Artifacts (Tracked)

This folder contains installable artifacts committed in git for simple download.

Current artifact:
- `vsc-codex-bridge-latest.vsix`

Install in VS Code:
1. Open Command Palette.
2. Run `Extensions: Install from VSIX...`.
3. Select `releases/vsc-codex-bridge-latest.vsix`.
4. Reload VS Code window when prompted.

After install:
1. Open your workspace in VS Code.
2. Set `vscCodexBridge.hubToken` in settings.
3. Run `VSC Codex Bridge: Start Bridge`.
4. Run `VSC Codex Bridge: Open PWA`.
